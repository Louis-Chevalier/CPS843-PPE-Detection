Go to the workspace and click on `Versions` tab

<img width="234" height="731" alt="image" src="https://github.com/user-attachments/assets/8e582673-34e6-4a04-a3c1-552e50435ef9" />

<img width="253" height="106" alt="image" src="https://github.com/user-attachments/assets/1c375044-8f54-4be5-889d-5f832a91f846" />

You can add `Preprocessing` and `Augmentations` to the dataset. The `Augmentations` step can easily double datasets since it copies and modifies images in certain way and then adds them back into the dataset.

<img width="887" height="931" alt="image" src="https://github.com/user-attachments/assets/23bc061f-443d-42cb-9fa1-033ee114e70e" />

Keep clicking on `Continue` and then `Create`. 

> [!IMPORTANT]
> Before you create this "Version" of the dataset, please ensure to add a simple message to decscribe the dataset in the `Version Notes` field.
